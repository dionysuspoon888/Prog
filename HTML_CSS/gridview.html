Grid-View

Many web pages are based on a grid-view, which means that the page is divided into columns (e.g., 12 columns of this page. Using a grid-view is very helpful when designing web pages. It makes it easier to place elements on the page.

This content area spans 9 columns and the left menu spans 3 columns. When you resize the screen, the width ratio of the content area and the menu keeps 9:3.
